# Python Learning Projects 

Collection of Python Projects built while learning programming fundamentals.

## Beginner Projects 
- Calculator : Basic arithmetic Operation
- Snake Water Gun : Rock-Paper-scissors variant
- Secret Code Translator : Python program to encode and decode messages with random characters and string manipulation
- Time Based Greeter: Python program that grets based on time
- KBC Game: CLI Quiz game inspired by KBC
- PinExtractor: Extracts PIN codes from a given text or dataset freecode camp  

## Intermediate Projects 
- UserConfig Manager : Practice dictionary CRUD operations and user settings management 
- Budget App: CLI app to track and manage personal budgets and expenses
- EmailSimulator : Simulates sending, reading, and deleteing emails in CLI
- MedicalDataValidator: validates medical dataset by checking formats, missing values, and data consistency
- RPG character: creates and manages RPG game characters with stats and attributes in CLI

## Learning Outcomes 
Most projects were created while following FreeCodeCamp's Python practice exercises, allowing me to apply concepts like 
functions,
loops,
file handling and OOP.
